# Security Policy

This is a static portfolio website. However, if you discover any security issues, please follow these guidelines.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you find a security vulnerability, please report it responsibly:

1. **Do NOT** create a public GitHub issue
2. **DO** email: ammar Yasir03062000@gmail.com
3. Include detailed steps to reproduce
4. Allow reasonable time for response before disclosing publicly

## Scope

This policy applies to:
- This portfolio website (AmmrYsir.github.io)
- Any associated web applications

## Out of Scope

- Social engineering attacks
- Physical security
- Denial of service attacks on this static site

## Response Timeline

- Initial response: Within 48 hours
- Resolution: Depends on complexity

Thank you for helping keep this project secure.
